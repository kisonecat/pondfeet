pondfeet

What is this?

With this tool, you can design a bitmap font by drawing some representative glyphs, and then let “deep learning” fill in the rest.

How do I use this?

Use your mouse to draw characters. As you draw, look to your right. Being somewhat optimistic, you will see a full character set respond to your stylistic choices.

This is somewhat aspirational to say the least…

How does this work?

I downloaded a large collection of bitmap fonts and then (hastily) trained a neural network in PyTorch to predict missing glyphs, given the shapes of the glyphs you provide. The model gets converted to ONNX so it can run in the browser via ONNX.js. There is code available if you want to find out more.

I'd appreciate your ideas on how to improve this. The design space for small bitmap fonts isn't that large so there are perhaps better ways to explore that space.
